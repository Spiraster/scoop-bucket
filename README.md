# scoop-bucket
This is my personal [Scoop](https://scoop.sh) bucket for apps that aren't present in the known buckets.

To use this bucket, add it to Scoop with: 

    scoop bucket add spiraster https://github.com/Spiraster/scoop-bucket.git
